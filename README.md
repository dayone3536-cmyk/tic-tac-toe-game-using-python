# tic-tac-toe-game-using-python
this a simple tic tac toe game it starts by taking the 2 players  input and stores them as X and O then it displays their chosen number over the board with their marker (X or O) then it checks if 3 of the same marker alings if yes it will print that current player won NOTE after a win it would just continue to run and to play just rerun the program
